Set-top box multimedia
How to use:
Click on one of the options bellow, write the rest of the command and click on "Send"
_________________________________________________________________________________________________________________________
[Name of command], [input on text box]			[Explanation of command]

Get object, <name of object> 				Prints the attributes and values of the object

Get object of type, <name of object type> 		Prints every object with the type requested (photo, video or film)

Get object that has, <string in object name> 		Prints every object or group with the given string in their name

Get group, <name of group> 				Prints the group and every object related to it

Create object, <type>|<name>|<filepath>|<attributes>|	Makes an object with the given attributes
Example: Create object, photo|gnuPicture|home/gnu.png|100|250|

Create group, <name of group>				Creates a group with the given name

Add to group, <name of group> <name of object>		Links an object to a group
Example: Add to group, group1 gnuPicture

Remove object, <name of object>				Remove an object with the given name

Remove group, <name of group>				Remove an group with the given name
